## Welcome to Quintet Mobile GitHub 

베타앱 배포를 위한 Public 페이지입니다.

### 배포 방식
1. 배포할 앱 설치 파일 준비 (ex: .apk, .aab, .ipa. .plist ...)
2. Dropbox에 앱설치 파일 공유
3. Dropbox에 공유한 파일 링크 copy
4. /Public/???/???.html 파일 링크 경로 수정
